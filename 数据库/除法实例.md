### 设置目标条件

设置如下关系：

Target（所在省市，是否自主划线）

| 所在省市 | 录取比例 |
| -------- | -------- |
| 北京市   | 25%      |
| 浙江省   | 10%      |
| 山东省   | 30%      |



### 运算结果

School(院校名称，是否自主划线)➗Target（所在省市，是否自主划线）
